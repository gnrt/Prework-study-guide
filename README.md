# Prework-study-guide

## Description

Basic study guide for Prework in the form of an HTML webpage with light JavaScript implemetation.
Created to familiarize with basic tools and procedures used in bootcamp and in industry.

## Usage

Take notes via HTML on topics covered in prework and categorize into subsections HTML, CSS, JavaScript and Git. Can inspect webpage and open console to see JavaScript running which will randomly generate a study topic to provide direction.
